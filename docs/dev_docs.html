Project Organization

Top Level Directories

Directory Description
docs Documentation files
images Images for html files (mostly unused)
lib External dependencies for project
plugins Scripts that define the language the user writes in
scripts Scripts for the user interface and programming environment
stylesheets Stylesheets for the project
tests Test suites for the project

Important Files

Filename Description
plugins/canvas.js Defines every block in the language
plugins/canvas_demo.js The demo programs
scripts/blocks.js Many utility functions for working with blocks
scripts/drag.js Drag and drop behavior
scripts/runtime.js Language runtime objects, like the Local and Global objects
scripts/workspace.js Sets up user interface; saving and loading functionality
